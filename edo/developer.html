Описание

REST API

Веб-сервис «REST API» – это HTTP-based API для обеспечения взаимодействия работы учетных систем с функционалом Clbs.Edo. Сервис позволяет реализовать следующие процессы:

отправка исходящих документов (одиночных и пакетных);

прием входящих документов (одиночных и пакетных);

обработка входящих документов (согласование, отклонение);

отправка и обработка запросов на аннулирование документов (входящих и исходящих);

Для данного веб-сервиса доступна спецификация OpenAPI. Для доступа к спецификации необходимо в браузере открыть страницу с адресом развернутого веб-сервиса. Подробнее см. Спецификация OpenAPI.
