## Типы иерархий

**IT\_СТРУКТ** - единая структура холдинга Руклимат

Под каждый объект создается отдельная иерархия

| Код символьный | Наименование   |
| -------------- | -------------- |
| 1C\_IZTT       | Структура ИЗТТ |
| 1C\_RT         | Структура РТ   |
| 1C\_TK         | Структура TK   |
| 1C\_VKO        | Структура VKO  |
| BIG            | Структура BIG  |
