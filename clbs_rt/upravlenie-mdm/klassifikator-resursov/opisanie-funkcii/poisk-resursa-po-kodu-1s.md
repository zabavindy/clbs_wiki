# Поиск ресурса по коду 1с

Заходим в «Классификатор ресурсов» нажимаем F2 и переходим в «1С УПП»

![](<../../../.gitbook/assets/2 (97).png>)

Выбираем «Поиск ресурсов по коду 1С». Вставляем код 1С. Нажимаем Ок

![](<../../../.gitbook/assets/3 (12).png>)
