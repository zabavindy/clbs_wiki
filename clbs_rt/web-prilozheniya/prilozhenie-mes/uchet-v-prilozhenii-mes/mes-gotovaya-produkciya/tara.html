Учет оборотной тары при отгрузке продукции

Если у рабочего центра указана настройка регистрации номера тары, то на форме будет доступно для ввода поле Код тары

Код тары вводится вручную или сканируется. После сохранения кода тары, отсканированный тип автоматически проставится в списке выбора тары

После ввода отгружаемого количества по каждой партии, можно сформировать документ отгрузки нажатием на соответствующую кнопку. Создается документ сдачи продукции на СГП, передается по стадиям бизнес-процесса
