# Тип техмаршрутов (PDM)

| Код  | Наименование типа техмаршрутов   |
| ---- | -------------------------------- |
| \_GT | Групповой техмаршрут             |
| \_RP | Исправление брака (групповой ТМ) |
| SMT  | Карты технического обслуживания  |
| ЕТ1  | Единичный техмаршрут             |
