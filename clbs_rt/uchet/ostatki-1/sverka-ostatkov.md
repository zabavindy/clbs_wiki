# Сверка остатков

Учет запасов -> Остатки (LV, 1C, ITE)

{% hint style="info" %}
Данные об остатках выводятся на текущий момент.
{% endhint %}

{% hint style="info" %}
Сверка остатков производится по двум основным складам РТ (Основной склад РТ и Основной склад СПР)
{% endhint %}
