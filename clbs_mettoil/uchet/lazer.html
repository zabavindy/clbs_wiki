Работа с шаблонами MaxiGraf

Требование к шаблону

Шаблон должен лежать в папке на рабочем компьютере MaxiGraf.   По умолчанию: C:\!backup\1 

Объект для QR-кода должен называться «ШтрихКод»

Объект для текста должен называться «Текст».

Привязка шаблона к ресурсу

Пункт меню:

Администрирование системы > Управление основными данными (MDM) > Ресурсы > Связь ресурса с именем файла шаблона

Для добавления привязки ресурса к шаблону необходимо перейти в пункт меню и нажать кнопку добавить (горячая клавиша F7):

для web-клиента

для обычного клиента

В открывшийся форме необходимо заполнить поля:

  • Код – уникальный код в справочнике;
  • Наименование шаблона в справочнике;
  • Имя файла – имя файла шаблона с расширением (.le);
  • Код ресурса – код ресурса, для которого применять данный шаблон

По окончанию ввода нажать «Сохранить»
