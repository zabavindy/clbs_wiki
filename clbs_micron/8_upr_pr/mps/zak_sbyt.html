Разбивка по паллетам

Введен и настроен тип норм 6SE Нормы использования технологической Тары

Для каждой продукции необходимо завести сколько штук на какую паллету (технологическая тара) помещается

На строчке Заказов, в Узле “Работа с паллетами” предусмотрено 2 RRT метода


1) “Очистить разбивку по паллетам” - Собственно говоря очищает имеющуюся разбивку по паллетам.

todo: если еще не было отгрузки

Если есть MPS план система не позволит очистить

2) “Разбить по паллетам”

Для каждой строки заказа ищет норму 6SE, и в зависимости от количества и нормы создает необходимо количество паллет типа Т, со сквозной нумерацией по заказу,
оставшиеся (где количества не хватило для полной укладки паллета) - укладывается на паллету типа С, без нумерации (Для дальнейшей обработки специалистом)


специалист путем ввода номера в строку указывает на какой паллете поедут остатки, так чтобы загрузка была наиболее эффективная,  из собственного опыта
 

 

Если нету нормы 6SE система выведет сообщение и добавит строку типа С
