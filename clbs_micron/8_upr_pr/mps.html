Для Запуска MPS Планирования необходимо нажать 

“Сформировать MPS план по заказам в производство”

 

В Результате в план попадут заказе, которых нету в MPS плане, по которым осуществлена разбивка по паллетам. (Если есть заказ со строками, но без разбивки по паллетам, то система уведомит и не даст сформировать MPS план).

Строки MPS плана создают строка к строке с разбивкой по паллетам в заказе

 

 

Также следует отметить, что ID строки разбивки храниться в расширении к таблице PLA_EXT в столбце ID_PAL_
